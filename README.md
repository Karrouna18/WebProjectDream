# DreamFlix Web Project

**Description:**  
Movie & Series Recommendation Website built using HTML, CSS, and JavaScript. The website allows users to search for movies, filter by categories, view movie details, and check actor information. It is designed to provide a user-friendly experience for discovering new movies and TV shows.

**Features:**  
- Search functionality by movie title and year  
- Category filtering and submenus  
- Movie modal displaying description, rating, and feedback  
- Star/Actor modal showing actor details and movies  
- Local storage implementation for user ratings and feedback  

**Technologies Used:**  
- HTML, CSS, JavaScript  
- Front-end only (no API integration)  
- Local storage for saving user input  

**Status:**  
Completed and fully functional as a front-end project.
